Features

Everything you need

A complete solution for converting .BAT and .CMD scripts to .EXE files with advanced customization options.

🔒
SHA-256 Password Protection

Password hashed with SHA-256 embedded in the EXE. Users get 3 attempts on each launch.

🎨
Custom Icon

Supports ICO, PNG and JPG formats. Automatic conversion to ICO using the Pillow library.

🏷
EXE Metadata

Embed product name, version, company, description and copyright directly into the executable.

📦
Embed / Wrapper Modes

Choose between Embed mode (script as resource) or Wrapper mode (script executed from memory).

⚡
UPX / LZMA Compression

Reduce EXE file size with UPX compression or a custom LZMA algorithm.

🌎
Polish & English

Full two-language support. Switch PL ↔ EN in one click without restarting.

🎭
5 Color Themes

Dark, Ocean, Sunset and more. Saved automatically in user configuration.

📋
Conversion Log

Detailed log of every operation with real-time PyInstaller console output preview.

How it works

Three simple steps

Converting BAT → EXE has never been easier. Zero configuration, zero technical knowledge required.

01
Select a .BAT file

Point to the .BAT or .CMD batch script you want to turn into an executable file.

02
Configure options

Add an icon, set a password, configure metadata and choose a compression mode.

03
Click Convert

PyInstaller packages everything into a single .EXE file. Ready for distribution.
